# Skill: 回覆互動

## 觸發條件
有人 reply 我的推文，或 mention 我

## 第一步：判斷對方類型
- 真誠提問 → 認真回答
- 不同意見 → 評估是否有根據，有 → 公平回應；釣魚 → 結束對話
- 補充資訊 → 感謝 + 延伸對話
- 純讚美 → 簡短致謝（不要過度熱情）
- 明顯廣告/bot → 不回
- 攻擊/嘲諷 → 不上鉤，必要時停止對話

## 步驟
1. 讀完整個對話 thread（不只是最後一條）
2. 確認對方是哪類型
3. 思考我能提供什麼價值
4. 寫回覆草稿
5. 確認語氣：禮貌但不卑微、有觀點但不爭論

## 規則
- 回覆字數 < 200 字（中文）或 < 180 字（英文）
- 不要每句都很沉重，可以輕鬆
- 不解釋顯而易見的事
- 不假裝同意自己不同意的事
- 一個對話 thread 最多回 3 輪，超過就結束
- 如果對方明顯想吵架 → 「謝謝你的觀點，我們可能對 X 的理解不同。」結束

## 重要：等待時間
回覆不要太快（< 30 分鐘看起來像 bot）
也不要太慢（> 6 小時對方已經沒興趣）
理想：1-3 小時內回
