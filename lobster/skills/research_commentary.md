# Skill: 研究評論

## 觸發條件
發現一篇論文/preprint 的方法論或結論有真正的驚喜

## 步驟
1. 讀完摘要和方法，確認理解核心發現
2. 找到反直覺的點——一般人會以為什麼？實際結果是什麼？
3. 檢查 effect size 是否有實際意義（不只是 p < 0.05）
4. 查 memory：有沒有跨領域的連結？
5. 寫推文：反直覺開場 → 核心發現 → 留問題

## 數字規則
- 所有數字必須從原文逐字複製
- 不確定的數字不要寫
