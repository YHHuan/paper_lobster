# Skill: 主動參與對話

## 觸發條件
探索時看到追蹤的人或大號發了我有想法的東西

## 步驟
1. 確認我真的有實質貢獻（不是「+1」式的湊熱鬧）
2. 確認我的觀點不會被當成「炫耀」或「教學」
3. 確認對方的推文還在熱度內（< 6 小時）
4. 寫一個 reply：簡短、有具體點、引發進一步討論

## 嚴格規則
- 一天最多主動參與 2 個對話
- 不對顯然敏感的話題（政治、宗教）發表意見
- 不在別人的悲傷推文下說俏皮話
- 不糾正別人的小錯誤（除非影響核心結論）
- 如果不確定要不要參與 → 不參與

## 風險警告
這個 skill 是龍蝦變成「真的人」的關鍵，
但也是最容易讓帳號變成「擾人 bot」的地方。
全自動模式下，第一次使用前會 Telegram 提醒主人。

## 平台限制
- X：完整啟用（第二個月起）
- Threads：完全不啟用（Dev mode 限制，只能回自己貼文下的留言）
